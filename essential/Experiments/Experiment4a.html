A Simple filter operation on a list is being done here.Based on some keyword entered in the input box , this filtering happens.

The ng-controller in the HTML code acts as the controller to with the model and the view will come to..This follows a Perfect MVC DEsign

                                    angular.forEach(arr, function(item){
		                         if (item.title.toLowerCase().indexOf(searchString) !== -1) {

		                             }
This will compare each word we type and compares it to the existing field and returns the filtered result. In this code , item item is every word we type and result is the final search phrase. This event happens after every keyup .
