Collecting the Form details

  • JJquery collects the value from the form and displays them separately
  • We get the value from the input box using the "('id').val()"
  • And we display the value as a text format using ".text()" function
  •                                       var c = $('#name').val();
                                        // this will set the value of name to variable c
                                      $('#name-d').text(c);
                                        // this will set the text value of "name-d" as the value of variable "c"
  • You can keep updating your values in the form and you will see the text value also getting updated on click of a button
