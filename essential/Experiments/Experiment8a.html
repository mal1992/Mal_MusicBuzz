Step 1

  • System Settings : MAC OS

  • Install node js from its website - Nodejs. Click on install and run the downloaded setup in you system
  • It will install npm - Node package manager along with it
  • Open your terminal and enter "echo $PATH" to check if its placed in /usr/local/bin
  • Now type "Node" which will start the session
  • Enter "console.log('Node Intro');" and see if the output is displayed as "Node Intro Undefined"
  • This means you have installed node successfully in your system
  • To exit from session , Press control + C 'Twice'
  • To install npm packages , just use "npm install (packagename)"

Step 2

  • Enter "npm install express" to install express package
  • Go to the node directory and create a js file called Serverapp.js and place the following code
  •                 var express = require('express')
                var app = express()
                app.get('/', function (req, res) {
                res.send('Node Introduction')
                })
                app.listen(3000)
  • now run this program in the terminal using node ./serverapp.js
  • the Port 3000 will be opened now
  • Open your favourite browser and go to "localhost:3000" and see if "Node Introduction" is displayed
  • Node has been successfully installed and now its time to create detailed apps
