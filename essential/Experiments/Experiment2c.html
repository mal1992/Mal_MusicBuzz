Documentation

  1. In this experiment we see how a table can be made responsive without any loss of data in the table
  2. The data is made to display one below the other to accomodate all the columns in a smaller screen
  3. This responsiveness will help in better display in smaller devices!

Code Snippet 

                div,table, thead, tbody, th, td, tr
             {
                display: block;
                border-radius:25px;
              }

When you make the display as block , then the table orientation is removed and it changes to become like a normal display
