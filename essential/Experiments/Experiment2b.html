Video with a Fluid Width

Documentation

This experiment deals with a fluid video

The Video is placed inside a iFrame and the iFrame is placed within a "div".I have made both the div responsive as well as the iframe

This Resizing of the Video will not interrupt any semantics. If the video is playing and the user resizes the screen , then the video wont face any disturbance.It will continue playing

Code Snippet

                           .vid-container iframe
                             { 
                              position: absolute; 
                              top: 0; 
                              left: 0; 
                              width: 100%;
                              height: 100%; 
                             }

In the Code snippet we notice we have a div with class name = vid-container and an iframe.On resizing the positioning becomes absolute and the fluid property is imparted. And since width =100% , it spans the entire conatiner it is present in.
