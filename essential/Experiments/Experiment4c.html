DOCUMENTATION

  • When you add an task , it gets added to the To-do List and the number of tasks pending will increase on top
  • When you check one of the tasks, then you will notice the count in the pending tasks reduce
  • You can add as much items you want to the list and keep striking them out as its done.
  •                                       $scope.remaining = function () {
                                        var count = 0;
                                         angular.forEach($scope.todos, function (todo) {
                                         count += todo.done ? 0 : 1;
                                          });
                                         return count;
    };
    This is the procedure which will keep track of the pending tasks to be done
