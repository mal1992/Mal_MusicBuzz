Documentation

This Experiment portrays how a grid can be made responsive based on the screen size. The main objective of this grid is to display all the contents present inside the grid on resizing. To avoid the grid being cut off on resizing , the grid elements are made to display one below the other for user comfort.

It will look better if the information is displayed in an orderly format and which spans the screen completely for better user experience.

Code Snippets

                @media only screen and (max-width: 480px) 
                {
                            .grid-col_2 { width:98%;border-radius:5px; }
                            .grid-col_3 { width:98%;border-radius:5px; }
                            .grid-col_4 { width:98%; border-radius:5px;}
                            .grid-col_5 { width:98%;border-radius:5px; }
                }

Media queries is used to impart responsiveness through CSS.

This code specifies that when the screen size falls below 480px, then the attributes for the grid columns specified also changes.
