Artist Reviews

COMPONENTS IN THIS PAGE

  • Navigation bar to move around the Website
  • Search Bar and Button
  • Review of the Artist entered
  • Footer - will navigate to HomePage

OUTLINE

On Entering any artist name, the latest top 20 reviews related to that Particular artist will be displayed. Only a Portion of the review will be displayed.A link will be provided so that the user can navigate to the parent page and read the complete review.

The review based on the artist is retreived by the API and the corresponding details from the response alone is displayed.

The link where the complete review is posted is also displayed
