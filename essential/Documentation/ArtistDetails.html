Artist-pedia

COMPONENTS IN THIS PAGE

  • Navigation bar to move around the Website
  • Search Bar and Button
  • Details of the Artist entered - From Wikipedia
  • Images of the artist
  • Videos of the artist - on click user will directed to the parent page containing website
  • Link to the artist Official Website
  • Link to related Pages of the Artist
  • Details related to the Artist
  • Related Artists based on the genre are also displayed
  • Footer - will navigate to HomePage

OUTLINE

Navigation bar to help users get around the website

There is a search bar which will allow users to enter artist name and search for their details

There is an image followed by some information about the artist from wikipedia. Most Specific details are only fetched from wikipedia.

There is an set of images of the artist that you can traverse through using the buttons.The images are randomly picked from websites which the API has reference to.

There is an set of videos of the artist that you can traverse through using the buttons.The videos are randomly picked from websites which the API has reference to.When you click on the play button , it will take you to the website where the video is hosted and will be played from there.

There is table where basic details of the artist as in when the artist began his career is present. Along with that in that table i have given links to that artists WikiPedia Page , His Official Page and his Twiiter Page.

There is a Table which will talk about how many albums he has sung and released.

Below is a section which will tell about the artist who are similar to that particular artist.It is chosen based on the genre they have sung in.

The next section will give details about the Genre what the artist has sung.It will give a list of genre that the artist has sung.

API USED

EchoNest API , Last.fm API , WikiPedia API are the API's that has been used in this Page.

The api call is made in the form specified below

             http://developer.echonest.com/*api_key*/v4/dynamic/create/artist_name=""

Wikipedia comes from MediaWiki API and is of the form

                 http://en.wikipedia.org/w/api.php
